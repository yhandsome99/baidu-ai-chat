# AI 智能对话助手

> 学号: 423830113 | 姓名: 浮标

基于**百度千帆大模型 ERNIE-4.0** 开发的智能对话桌面应用，使用 Python + Tkinter 实现，可在 PyCharm 中直接运行。

## 功能特性

| 模式 | 功能说明 |
|------|---------|
| 智能问答 | 回答各类知识性问题 |
| 文本创作 | 写作、故事、诗歌创作 |
| 代码助手 | 编程辅助、代码调试、逻辑解释 |
| 翻译助手 | 中英文互译 |
| 情感陪伴 | 温暖陪伴、倾听交流 |

## 技术栈

- **语言**: Python 3.8+
- **界面**: Tkinter（内置，无需安装）
- **AI 模型**: 百度文心 ERNIE-4.0
- **API**: 百度千帆大模型平台 v2 接口
- **开发工具**: PyCharm

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 在 PyCharm 中运行

直接打开 `main.py`，点击右上角运行按钮即可。

### 3. 命令行运行

```bash
python main.py
```

## API 说明

本项目调用百度千帆大模型平台接口：

- **接口地址**: `https://qianfan.baidubce.com/v2/chat/completions`
- **模型**: `ernie-4.0-8k`
- **认证方式**: Bearer Token（BCE v3 API Key）
